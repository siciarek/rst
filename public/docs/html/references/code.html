10.4. Uwagi¶

W przypadku Twig-a blok .. code-block:: twig nie działa.

.. code-block:: twig

    {{ 'John' ~ ' ' ~ 'Doe' ~ ' ' ~ date('Y-m-d') }}

Należy użyć lexera jinja.

{{ 'John' ~ ' ' ~ 'Doe' ~ ' ' ~ date('Y-m-d') }}

.. code-block:: jinja

    {{ 'John' ~ ' ' ~ 'Doe' ~ ' ' ~ date('Y-m-d') }}

A w przypadku kodu zagnieżdżonego w HTML .. code-block:: html+jinja.

<h1>{{ 'John' ~ ' ' ~ 'Doe' ~ ' ' ~ date('Y-m-d') }}</h1>

<ul>
{% for a in 1..10 %}
    <li>{{ a ~ '. ' ~ 'John' ~ ' ' ~ 'Doe' ~ ' ' ~ date('Y-m-d') }}</li>
{% enfor %}
</ul>

    <h1>{{ 'John' ~ ' ' ~ 'Doe' ~ ' ' ~ date('Y-m-d') }}</h1>
